# Effect Crystals
Make cool crystals that give effects! You can also upgrade them!

## Effect Colors
- Red - **Strength**
- Orange - **Fire Resistance**
- Yellow - **Haste**
- Lime - **Jump Boost**
- Light Blue - **Speed**
- Dark Blue - **Water Breathing**
- Pink - **Regeneration**
- Grey - **Resistance**
- Black - **Night Vision**
- White Outline - **Tier 2 Crystal** *(Gives the effect but Level 2)*

If you need help, use `/ec_help all` in game.
